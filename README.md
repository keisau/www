## tech stack

1. Vue.js
  1. Vue router
  2. Prerender SPA Plugin
2. Webpack 2
3. Babel (ES2015 - 2016)

5. PostCSS
6. SCSS
7. Font-awesome

2. AWS S3
3. Cloudflare
